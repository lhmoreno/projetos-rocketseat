# Daily Diet API

## Iniciar o projeto
Intale as dependências usando:
```bash
npm install
```

Inicie o banco de dados:
```bash
npm run knex -- migrate:latest
```

Rode o projeto com:
```bash
npm run dev
```

## Rotas
Todas as rotas estão descritas em um arquivo do [Insomnia](https://insomnia.rest/) em [insomnia.json](./insomnia.json)
